# Verde Analytics Lab

Aplicacion interactiva en Streamlit para visualizacion de distribuciones
de probabilidad y prueba de hipotesis Z, con asistente IA (Google Gemini).

## Modulos

* Carga de datos CSV o generacion sintetica
* Analisis exploratorio: Histograma, KDE, Boxplot, Q-Q Plot
* Prueba Z con region critica visualizada
* Asistente Gemini para validar conclusiones

## Instalacion

pip install -r requirements.txt
streamlit run app.py





\## Uso de IA (Google Gemini)

Para el desarrollo de este proyecto utilicé la IA de Gemini para ayudarme a estructurar el análisis de las variables y generar la interpretación automática de la Prueba Z.
